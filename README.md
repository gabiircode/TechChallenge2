## 📊 Visão Geral do Projeto

Este projeto faz parte do **Tech Challenge - Fase 2** da pós-graduação em **Data Analytics**. O objetivo principal é desenvolver um modelo preditivo capaz de prever se o índice IBOVESPA irá fechar em **alta (↑)** ou **baixa (↓)** no dia seguinte.

Este modelo serve como um insumo crucial para alimentar dashboards internos de tomada de decisão de analistas quantitativos em um grande fundo de investimentos brasileiro.
